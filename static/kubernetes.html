Why Kubernetes?

Kubernetes is an open-source container orchestration platform. It helps manage containerized applications at scale by handling deployment, scaling, and availability.

Use in This Project

In this project, we define Kubernetes manifests to deploy our Dockerized web application to a cluster, allowing automated and scalable application management.
